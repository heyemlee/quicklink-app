做一个后端，提供后台管理。
为每个客户创建一个账号，登录后可以在后台自己修改：
公司名称 / Logo, 电话、地址, 社交媒体链接, 配色 / 样式, 展示的模块(联系、关注、评价等)

✅ Next.js App Router (动态路由 + SSR/ISR)
✅ 数据库：Prisma + PostgreSQL
✅ 认证系统：NextAuth.js
✅ 后台管理界面

功能模块拆解:
1.登录功能/login ， /dashboard
2.后台编辑页面（客户自己改内容）
3.保存到数据库
4.动态生成名片页面 (/card/[slug]/page.tsx
)

开发顺序建议

1️⃣ 数据库 + Prisma + 用户表建好
2️⃣ 实现 /card/[slug] 动态页面 (先写死一个用户测试)
3️⃣ 实现 /dashboard 编辑基本信息 → 保存到数据库
4️⃣ 绑定 slug，让每个用户访问自己的页面
5️⃣ 增加社交链接、自定义配色等功能
6️⃣ 打磨 UI & 体验（组件化 + 可复制）
